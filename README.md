# 💻 Online Code Judge System

## Overview

Online Code Judge System is a web-based coding platform that enables users to write, execute, test, and submit programming solutions in real time. The platform automates code execution and evaluation while maintaining submission records and competitive leaderboards.

Designed for coding assessments, technical interviews, and competitive programming environments, the application provides a seamless experience for both learners and evaluators.

## API Endpoints

### Code Execution

POST /run

Execute submitted code and return output.

### Submission

POST /submit

Store user submission and score.

### Leaderboard

GET /leaderboard

Retrieve ranked participant scores.
